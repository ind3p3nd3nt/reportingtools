# reportingtools
Bash scripts to do various reports and scans given an ip list
# installing 
```
git clone https://github.com/ind3p3nd3nt/reportingtools;
cd reportingtools;
sh install.sh
```
# running
```
sh whois.sh >>whois.log 
sh traceroute.sh >>traceroute.log
sh nmap.sh >>nmap.log
```
